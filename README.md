## Requirements

* Nodejs 4+

## Installation

Execute `npm install` to install the application

## Features

* Local server development
* Livereload
* ES6 syntax (Babel)
* ReactJS with Flux workflow
* Material UI design
* Eslint

## Commands

### `npm run start`

Start server as development mode with livereload
